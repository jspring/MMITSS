Description:
This plugin was created  as a demonstration of how to create a class via the Plugins SDK.  This code is provided as-is and there is no promise of it being fit for production.  However, it should be very explanatory and be a good learning example of how to create a class.  If you find any bugs or have suggestions, please report them feedback system.

Topics this project covers:
How to make a plugin class
 How to make Properties (regular, computed and shared)
 How to make Methods (regular and shared)
 How to make Constants
 How to expose an Event
 How to implement an Interface
 Console safety
